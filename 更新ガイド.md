# Iry -アイリー- 公式サイト 更新ガイド

このサイトは **`js/site-data.js` という1つのファイルを編集するだけ** で、
写真・ブログ・料金・Instagram・お知らせなど、ほぼすべての内容を更新できます。
HTMLやCSSの知識は不要です。

編集には Windows標準の「メモ帳」でもOKですが、
無料の **VS Code**(https://code.visualstudio.com/)を使うと間違いに気づきやすくおすすめです。

---

## 最初にやること(開業前チェックリスト)

`js/site-data.js` を開いて、上の方にある `info:` の中を実際の情報に書き換えてください。

- [x] `tel` — 店舗の電話番号(096-247-9488 反映済み)
- [x] `lineUrl` — 公式LINE(https://lin.ee/Wo5OaPz 反映済み)
- [x] `squareUrl` — Square予約ページURL(反映済み。変更する場合はここを書き換え)
- [x] `instagramUrl` — Instagramプロフィール(反映済み)
- [x] お客様の声 `voices:` — Googleクチコミ3件反映済み(追加はいつでも可能)
- [ ] `hours` / `holiday` — 営業時間と定休日(現在は仮の内容です)
- [ ] `legal.html` を開き、「●●」の箇所(事業者名・責任者・メールアドレス)を書き換える
- [ ] 料金の `partPrice` / `parts` / `comboExamples` / `valueSets` は現在の想定価格です。実際の金額と相違ないか確認してください

## 写真を追加する

1. 写真ファイルを `images/gallery` フォルダに入れる(例: `shop-1.jpg`)
2. `js/site-data.js` の `gallery:` に1行追加する

```js
gallery: [
  { file: "images/gallery/shop-1.jpg", caption: "施術ルーム(完全個室)" },  // ← こういう行を追加
  ...
],
```

- 写真がまだ無い項目は自動で「PHOTO」というプレースホルダーが表示されます
- 写真は横長(4:3)が一番きれいに表示されます
- **お客様が写る写真は必ず掲載許可(モデルリリース)を取ってから**

スタッフ写真は `images/staff` フォルダに入れて、`staff:` の `photo` を書き換えます。

---

## ブログを追加する

`js/site-data.js` の `blog:` の **一番上** に、以下の形をコピーして追加します。

```js
blog: [
  {
    date: "2026-08-01",
    category: "お知らせ",
    title: "記事のタイトル",
    body: "<p>1つ目の段落。</p><p>2つ目の段落。</p>",
  },
  // ↑ 新しい記事はここ(一番上)に。古い記事の上に積んでいくイメージです
```

- 段落は `<p>〜</p>` で囲む/段落内で改行したいときは `<br>`
- トップページには自動で最新3件が表示され、`blog.html` に全記事が並びます
- 医療効果の断定表現(「必ず生えなくなる」等)は薬機法上NGです。「個人差があります」を添えてください

## お客様の声(Googleクチコミ)を反映する

1. Googleビジネスプロフィールの管理画面を開く → 「クチコミ」
2. 掲載したいクチコミを選び、本文をコピー
3. `js/site-data.js` の `voices:` の中の `text` に貼り付け、`meta` に星の数などを入れる

```js
voices: [
  {
    meta: "Googleクチコミ ★★★★★",
    title: "見出し(クチコミの内容から一言でOK)",
    text: "実際のクチコミ本文をここに貼り付け",
  },
  ...
],
```

- 現在は「準備中」のプレースホルダーが3件入っています。実際のクチコミに差し替えてください
- 個人が特定されすぎる情報(フルネーム等)は避け、Googleの表示名のままで問題ありません

## Instagramの投稿をサイトに載せる

1. Instagramで載せたい投稿を開く
2. アドレスバーのURLをコピー(例: `https://www.instagram.com/p/AbCdEfG/`)
3. `js/site-data.js` の `instagramPosts:` に貼り付ける

```js
instagramPosts: [
  "https://www.instagram.com/p/AbCdEfG/",
  "https://www.instagram.com/p/HiJkLmN/",
],
```

- 行頭の `//` を消すとその行が有効になります(`//` はメモ扱い)
- 投稿を非公開/削除するとサイト側も表示されなくなるので、URLも消してください

## 多言語ページ(英語・繁体中文)について

SEO対策として、トップページの英語版・繁体中文版を用意しています。

- `index.html` — 日本語(通常運用はこちらのみでOK)
- `index-en.html` — English
- `index-zh.html` — 繁體中文(台湾向け)

ヘッダー右上の「JA / EN / 中文」から切り替えられます。

**重要**: 英語版・繁体中文版は `js/site-data.js` とは別の
`js/site-data.en.js` / `js/site-data.zh.js` という専用ファイルを使っています。
これは今の内容を翻訳した「スナップショット」なので、**日本語の料金や営業時間を
変更しても、英語・中国語版には自動で反映されません。** 金額や営業時間など
重要な情報を変えたときは、翻訳サービス(DeepL等)やご自身の英語・中国語力で
`site-data.en.js` / `site-data.zh.js` の該当箇所も合わせて書き換えてください。

ブログ記事・特定商取引法のページ・Googleクチコミ本文は、英語版・中国語版でも
日本語のまま表示されます(翻訳の維持が大変なため、意図的に日本語のままにしています)。

## ボタンのリンク先を変える

サイト内のボタンは、**リンク先を `js/site-data.js` の `info:` に1回書くだけで、
同じ種類のボタン全部にまとめて反映**される仕組みです。ボタンを1個ずつ直す必要はありません。

| ボタン | site-data.js のどこを書き換えるか |
|---|---|
| 「予約する【最新脱毛メニュー】はこちら」(青いボタン全部) | `squareUrl` |
| 「LINEで相談する」(緑のボタン全部) | `lineUrl` |
| 「電話で予約する」 | `tel` |
| 「@iry をフォローする」 | `instagramUrl` |

例: Squareの予約ページを作り直してURLが変わったら、

```js
squareUrl: "https://book.squareup.com/appointments/(新しいURL)",
```

の1行を書き換えて保存すれば、サイト内の予約ボタンすべてが新しいURLになります。

## スタッフ写真を入れる(Instagramの写真を使う場合)

1. スマホのInstagramアプリで自分のプロフィールを開く
2. プロフィール写真を長押し(または投稿から使いたい写真を選ぶ)→ スクリーンショットや「写真に保存」で端末に保存
3. 保存した写真をパソコンに送り、ファイル名を `owner.jpg` にして `images/staff` フォルダに入れる

これだけでスタッフ紹介に表示されます(正方形に近い写真がきれいに表示されます)。

## 顔の施術部位カード(実写写真+光るゾーン)について

料金セクションの「顔の施術部位」は、実際の施術写真の上に照射範囲を
青い光(#00D9FF)で重ねて表示する仕組みです。

- 写真: `images/face-parts` フォルダの8枚(odeko/hoho/hana/hanashita/kuchishita/ago/momiage/agoshita)
- 表示範囲や光る形: `js/face-zones.js` の `ZONES` で管理
- 部位名と価格は `site-data.js` の顔パーツ(順番どおり)から自動で取得されるため、
  価格を変えるときは今までどおり `partPrice` を書き換えるだけでOK
- **写真を差し替える場合**: 同じファイル名で `images/face-parts` に上書き。
  構図が変わる場合は光る範囲(ZONESのpath)の調整が必要なので、その際はご相談ください
- WordPressなど他のサイトへ移植する場合: `js/face-zones.js` + `images/face-parts` +
  `<div id="face-zones"></div>` の3点を持っていくだけで動きます

## 料金を変更する(パーツ制)

料金は「S・L・Xの3段階×部位」で決まる仕組みです。`js/site-data.js` の中に4つの項目があります。

- `partPrice` — S/L/Xそれぞれの単価(ここを変えると全部位の価格に自動反映されます)
- `parts.face` / `parts.body` — 部位の一覧と、各部位がS/L/Xのどれかの設定
- `comboExamples` — トップページの「組み合わせ例」カードの中身(合計金額は手入力なので、`partPrice`を変えたらここも書き換えてください)

部位を増やす/減らすときは、`parts.face.items` または `parts.body.items` に
`{ name: "部位名", size: "S" }` の行を足し引きします(sizeは "S" "L" "X" のいずれか)。

※ 料金表の横の人体図は、部位名が site-data.js と完全に一致している場合に
表と連動して光ります。部位名を変えた場合、図の点は光らなくなりますが表示には支障ありません。

## よくある失敗と直し方

| 症状 | 原因 |
|---|---|
| ページが真っ白/一部消えた | `site-data.js` のどこかで `"` や `,` が欠けている。直前に編集した箇所を確認 |
| 文字化けする | ファイルを「UTF-8」で保存し直す(メモ帳なら保存時に選べます) |
| 写真が「PHOTO」のまま | ファイル名の綴り違い(大文字小文字も区別されます)、またはフォルダが違う |
| Instagramが表示されない | URLが投稿のURLではなくプロフィールのURLになっている |

編集前に `site-data.js` をコピーしてバックアップを取っておくと安心です。

## サイトの公開について

このサイトはHTMLだけで動く「静的サイト」なので、どのレンタルサーバーでも公開できます。
おすすめは無料の **Netlify**(https://www.netlify.com/)か **Cloudflare Pages**:
フォルダをドラッグ&ドロップするだけで公開でき、更新時も同じ操作です。

公開後にやること:

1. **Googleビジネスプロフィール**にサイトURLを登録(「ウェブサイト」欄)
2. **Instagram・LINEのプロフィール**にサイトURLを設定
3. Googleサーチコンソールに登録して検索状況を確認

```
├── legal.html        … 特商法・ポリシー(開業時に●●を書き換え)
├── 更新ガイド.md      … このファイル
├── css/style.css     … デザイン(編集不要)
├── js/
│   ├── site-data.js  … ★ 更新はこのファイルだけでOK ★
│   ├── main.js       … (編集不要)
│   └── hero-video.js … トップの動画演出(編集不要)
├── media/
│   └── hero/         … トップの動画4本(ヒーロー1本+証拠動画3本。差し替え方法は上記)
└── images/
    ├── before-after/ … ビフォーアフター写真(site-data.jsのgalleryで管理)
    ├── gallery/      … その他の店内写真用フォルダ
    └── staff/        … スタッフ写真を入れる
```

### トップの動画について

トップページは「照射動画のヒーロー」+「スクロールで順に現れる証拠動画3本」の構成です。

- **ヒーロー動画(照射)を差し替える**: `media/hero` に動画を入れ、`js/hero-video.js` の
  `HERO_VIDEO` のファイル名を書き換える
- **証拠動画(抜けた肌/ジェル/足)を差し替える・並べ替える**: `js/site-data.js` の
  `proofVideos:` を編集する。`file` がファイル名、`title` が縦書きの見出し、`text` が説明文。
  順番を入れ替えたい時は `{ }` ごと上下を入れ替えるだけです
- 動画は1本5〜10MB程度に圧縮したものを使ってください(現在の4本は圧縮済み・各6〜7MB)
